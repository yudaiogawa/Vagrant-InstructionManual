About
--------
* Immutable Infrastructure as Code を実現する
* 仮想マシンを構築するためのツール
* 開発環境を簡単に構築できる
* 設定を一度記載すれば、何回でも同じ構成の仮想マシンを立ち上げれる。
* プロビジョニング機能
