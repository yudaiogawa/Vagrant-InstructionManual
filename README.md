# Vagrant ;)

## Overview

- Immutable Infrastructure as Code
- 開発環境の構築および共有を簡単に行うためのツール
- 開発環境構築の自動化
- 設定を一度記載すれば、何回でも同じ構成の仮想マシンを構築できる
- プロビジョニング機能を備えている

## Required Tools

- [Vagrant](http://www.vagrantup.com)
- [VirtualBox](http://www.virtualbox.org/)
